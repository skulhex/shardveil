# Shardveil

Учебный проект: тёмный roguelike в сеттинге упавшего под землю мира.  
История о рыцаре, ищущем свет среди древних катакомб и шепчущих руин.

Проект находится на раннем этапе разработки.  
Реализованы базовые сущности: Player и первый враг Skeleton.

## Структура проекта
```
shardveil/
├── src/ 
│   ├── main.py         # основной игровой цикл
│   └── sv/          # папка с игровыми модулями
│       ├── core/       # настройки проекта, константы
│       └── entities/   # классы сущностей
└── assets/             # папка с графическими ресурсами
```

## Установка и запуск
Склонируйте репозиторий и установите зависимости через pip:
```
git clone https://github.com/sirex0101/shardveil
cd shardveil
pip install -r requirements.txt
```

Запуск игры:
```
python src/main.py
```
